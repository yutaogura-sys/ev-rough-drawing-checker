# ラフ図チェックツール

EV充電設備 補助金申請用に、工事会社が作成する**ラフ図・現地調査報告書・要部写真**が、作図センターで補助金申請用図面（設置場所見取図／平面図／配線ルート図／電気系統図）を正確に作図できるレベルかを判定するチェックツールです。

## 🌐 使い方（ブラウザでそのまま動きます）

▶️ **[こちらから起動](https://yutaogura-sys.github.io/ev-rough-drawing-checker/)**

インストール不要・ログイン不要・サーバー送信なし（全てブラウザ内で完結します）。
入力内容は自動でブラウザの localStorage に保存されます。

## ✨ 特長

### 📋 4図面ごとの完全網羅チェック
- ① 資料受領（ラフ図／現調報告書／要部写真）
- ② 設置場所見取図（敷地・公道・案内板・充電スペース）
- ③ 平面図（充電スペース寸法・基礎・車止め・防護用部材）
- **★④ 配線ルート図（最重要・29項目）★**
- ⑤ 電気系統図（配電方式・ブレーカ・接地）

### ⚡ 配線ルート図を特に重視
工事会社が作成する資料の中で最も難しい「配線ルート図」を正確に作図できるよう、29項目のチェックを用意しています：

- 電源取得元（既設盤の特定／空きブレーカ／主幹容量）
- 配線ルート全体の追跡可能性
- 各区間の横引き距離・立上げ・立下げ
- 線種（CV8sq-3C／CVT100sq／VVF2mm-3C 等）
- 配管種別（PFD-28／HIVE-42／E25／FEP30 等）
- 配線方法（露出／埋設／架空／天井内／壁内）と路面状況
- プルボックス位置・寸法
- 障害物（柱・梁・既設配管・空調等）の迂回情報
- 高低差・斜面・スロープの立上げ立下げ換算
- 数量妥当性（過度な余長の検出）

### 💰 補助金料金監視機能
- 補助金上限額に対する想定工事費の使用率を可視化
- **80%接近で警告**
- **超過時に「あなたは ¥◯◯ 分オーバーしています」と明示**
- 区間ごとの配線長を作図センター標準値（NEV審査基準）と自動照合し、過度な余長（補助金不正受給を疑われるレベル）を赤色警告

### 🎯 自動判定
- ✅OK / ⚠️一部 / ❌不備 / －該当なし の4段階入力
- 重要度別の重み付け（critical=5, high=3, medium=2, low=1）
- 必須項目に1つでも❌があれば自動的に「差戻し」判定
- 配線ルート図と総合スコアの両方が90%以上で「作図可」

### 📤 出力形式
- 印刷 / PDF保存（ブラウザの印刷機能経由）
- JSONエクスポート / インポート（案件データの保存・復元）
- テキストでクリップボードにコピー

## 🛠 技術仕様

- 単一HTMLファイル（CSS・JavaScript埋め込み）
- 外部依存ゼロ・サーバー通信ゼロ
- すべてのモダンブラウザに対応（Chrome / Edge / Safari / Firefox）
- データはブラウザの localStorage に保存

## 📚 準拠資料

本ツールは以下を参考に設計しています：

- 交付申請図面作成マニュアル（作図センター）
- 交付申請図面における配線配管長の考え方（R8-1基準）
- EV充電 標準設計仕様書 Ver4.0
- 設置場所見取図／平面図／配線ルート図／電気系統図 の補助金要件（NEV基準）
